=================================================
MindVault – Design System (Project 1)
=================================================

FONT
-------------------------------------------------
Primary Font:
- Inter (Google Fonts)

Font Weights:
- 400 → Body text
- 500 → Labels, buttons
- 600 → Section headings
- 700 → App title (MindVault)

Font Family (Tailwind):
- font-sans → Inter, sans-serif


TYPOGRAPHY SCALE
-------------------------------------------------
App title        : text-xl / text-2xl
Section heading  : text-lg
Note title       : text-base font-semibold
Note content     : text-sm
Meta text (date) : text-xs


COLOR SCHEME
-------------------------------------------------

PRIMARY COLORS
-------------------------------------------------
Header Background
- Tailwind : bg-blue-900
- Hex      : #1E3A8A

Primary Button / CTA
- Tailwind : bg-blue-700
- Hex      : #1D4ED8

Primary Button Hover
- Tailwind : hover:bg-blue-800
- Hex      : #1E40AF


NEUTRAL COLORS (MAIN UI)
-------------------------------------------------
App Background
- Tailwind : bg-gray-100
- Hex      : #F3F4F6

Card Background
- Tailwind : bg-white
- Hex      : #FFFFFF

Borders / Dividers
- Tailwind : border-gray-200
- Hex      : #E5E7EB


TEXT COLORS
-------------------------------------------------
Primary Text
- Tailwind : text-gray-900
- Hex      : #111827

Secondary Text
- Tailwind : text-gray-600
- Hex      : #4B5563

Muted / Placeholder Text
- Tailwind : text-gray-400
- Hex      : #9CA3AF


ICON COLORS (Lucide Icons)
-------------------------------------------------
Default Icon
- Tailwind : text-gray-500

Active / Emphasis Icon
- Tailwind : text-blue-700

Destructive Icon (Delete)
- Tailwind : text-red-500


DESIGN RULES (LOCKED)
-------------------------------------------------
- Use ONLY the above colors
- No neon or extra accent colors
- No heavy shadows
- Icons use currentColor
- SVGs have fill="none" and stroke="currentColor"
- Calm, minimal, readable UI
- No visual noise


PROJECT
-------------------------------------------------
Project Name : MindVault
Stack        : React + Tailwind
Icons        : Lucide (SVG / components)
Font         : Inter
Style        : Simple, clean, professional
